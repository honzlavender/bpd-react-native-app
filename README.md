# A Crisis Kit 
a distraction app for panic & anxiety attacks, episodes, and impulse control

## Getting Started

Fork and/or download zip code and open in local IDE

### Prerequisites

- Node.js
- Git
- Watchman

And suggested install of Expo Simulator

```
npm install --global expo-cli
```
